### Installation

1. git clone the project

```bash
git clone

```

2. Navigate inside the project directory

```bash
cd issue-tracker

```

3. install dependencies

```bash
npm i

```

4. Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
